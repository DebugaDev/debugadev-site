Algumas coisas sobre Java — II

Dando continuação aos meus estudos, escrevi mais um pouco sobre coisas básicas. A ideia agora é entender um pouco sobre os tipos de dados em Java.

As linguagens de programação permitem a criação de variáveis para manipular informações, exemplo: posso guardar a idade de uma pessoa em uma variável do tipo inteiro ou o valor de venda de um produto em uma variável do tipo double.

Em Java, quando precisamos de uma nova variável se torna necessário declará-la. A declaração deve informar o o TIPO e o NOME dessa variável. Já ouviu falar em “fortemente tipada”? É por isso, todas as variáveis devem ter seu tipo definido na declaração.

Exemplo de declaração: 

    
      

        public int idade;

Também é importante falar que em Java a inicialização das variáveis é obrigatória e pode ocorrer de duas maneiras: explicitamente e implicitamente.

Variáveis locais devem ser inicializadas de maneira explícita antes de serem utilizadas, exemplo: 

    
      

        public void fazAlgo() {
          int idade = 25;
          System.out.println(idade);
          return hello;
        }

O fato aqui é que a variável tem que ser inicializada antes de seu uso, isso pode ocorrer de outras maneiras, mas caso não seja inicializada ocorre um erro de compilação. E se a variável for atributo de uma classe? Ela é inicializada de maneira implícita junto com o objeto da classe com valor padrão.

Exemplos de valores padrão:

  • Inteiros: 0
  • Ponto flutuante: 0.0
  • Boolean: false
  • Char: vazio, equivalente a 0
  • Referências: null

Tipos Primitivos

No Java todos os tipos primitivos já estão definidos, não sendo possível criar novos tipos primitivos. Os tipos primitivos são: byte, short, char, int, long, float, double e boolean.

Pontos legais de se saber:

  • boolean é o único não numérico
  • float e double são pontos flutuantes
  • os demais são inteiros
  • Os números de ponto flutuante podem assumir os seguintes valores: +/- infinity, +/- 0, NaN (Not a Number)

Tipos e seus tamanhos

  • byte 1 byte (8 bits, de -128 a 127)
  • short 2 bytes (16 bits, de –32.768 a 32.767)
  • har 2 bytes (só positivo),(16 bits, de 0 a 65.535)
  • int 4 bytes (32bits, de –2.147.483.648 a 2.147.483.647)
  • long 8 bytes (64 bits, de –9.223.372.036.854.775.808 a 9.223.372.036.854.775.807)
  • float 4 bytes (32bits, de +/– 1.4 * 1⁰⁴⁵ a +/– 3.4028235 * 1⁰³⁸)
